# 📝 To-Do List (Java)

A simple console-based **To-Do List application** built using **Java** that allows users to manage daily tasks efficiently.

This project demonstrates basic concepts like **loops, condition handling, and ArrayList operations**.

---

## 🚀 Features

* 📋 View all tasks
* ➕ Add new tasks
* ❌ Delete tasks by number
* 🔄 Real-time task updates
* 🧠 Menu-driven console interface

---

## 🛠️ Tech Stack

* **Java**
* **ArrayList (Collections Framework)**
* **Scanner Class (User Input)**

---

## 📂 Project Structure

```id="code9"

│── ToDoList.java   # Main program
```

## ⚙️ How It Works

* Tasks are stored in an `ArrayList`

* User interacts through a menu system

* Tasks can be:

  * Added
  * Viewed
  * Deleted

* Each task is indexed for easy deletion

## ⚠️ Limitations

* No data persistence (tasks are lost after exit)
* No task deadlines or priorities
* Console-based UI
